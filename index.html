INTODESHALLAH välkommen till mitt API


Routerna

🔒 GET/users - returnerar alla användare som finns i databasen. ID, name och username medföljer

🔒 GET/users/{id} - returnerar en användare med specifikt id, samma värden som i users routen medföljer

POST/users - skapar en ny användare och accepterar JSON object på formatet {"username", "password", "name"}

🔒 PUT/users/{id} - ändrar en befintlig användare med specifikt id och accepterar JSON object med formatet {"username", "password", "name"}

POST/login - tar JSON object med formatet {"username", "password"} och jämför det med databasen. Ifall infon matchar med en användare som finns i databasen loggas man in och får en bearer token som behövs för att komma åt alla låsta router

🔒 = kräver en giltig bearer token i authorization header
